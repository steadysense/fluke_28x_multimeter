============
Installation
============


Stable release
--------------

To install Fluke 28x Multimeter, run this command in your terminal:

.. code-block:: console

    $ pip install fluke_28x_multimeter

This is the preferred method to install Fluke 28x Multimeter, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Fluke 28x Multimeter can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mofe23/fluke_28x_multimeter

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/mofe23/fluke_28x_multimeter/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/mofe23/fluke_28x_multimeter
.. _tarball: https://github.com/mofe23/fluke_28x_multimeter/tarball/master
